Навыки

Что я использую в работе

HTML5
Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта
CSS3
Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!
Java Script
Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое
Jquery
Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект я её не буду, но навык работы с ней присутствует
React
Эта библиотека позволяет создавать web-приложения. Я могу создать максимально интерактивный продукт именно под ваши цели
Bootstrap
Фреймворк Bootstrap позволит ускорить разработку. С помощью этого инстурмента я могу в короткие сроки сделать проект
Gulp
С помощью этого таск-менеджера я ускоряю свою работу и трачу меньше времени на оптимизацию проекта
GitHub
Система контроля версий Git, позволит мне хранить все версии проекта и всегда вернуться к прошлой если это нужно
Webpack / Parcel
С помощью этого сборщика модулей я могу создавать проекты с разборчивой структурой и настраивать проект до малейших деталей.
Brain
С помощью этого инструмента я способен изучать все что нужно для работы
